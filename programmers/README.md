## **Files**

### **LEVEL1**
프로그래밍 입문자를 위한 쉬운 수준이라고 명시되어 있지만, 카카오에서 출제 된 문제들은 쉽지많은 않다. <br/>
문제를 잘 이해하고, 필요한 자료구조를 사용해야 한다. <br/>
어려운 자료구조나 알고리즘(ex. 트리, 이분탐색트리, DFS, BFS, 동적 계획법등..) 나오지는 않지만, 기본적으로 배열, 스택, 큐에 대한 이해와 적절한 사용이 필요한 문제들이 있다.
